# orbition_bot
项目地址：https://airdrop.orbition.network?ref=3976686569

Orbition Network（orbition.network） 是一个早期阶段的 AI-Oriented（AI 导向） 区块链网络，专注于去中心化 AI 基础设施、用户参与和生态建设。核心代币为 $OBN，项目强调早期贡献者参与（如 AI Mining）、未来 staking、节点运行和移动端体验。

项目核心特点定位：EVM 兼容的 Layer-1 区块链，灵感来源于 Solana 等项目（支持 DPoS 共识、EVM/SVM/zKEVM 等），目标是构建支持去中心化 AI 的网络。
参与方式：AI Mining：浏览器/网页挖矿（当前速率约 0.01 OBN/秒），连接钱包即可启动。
节点/验证者运行（Testnet）。
任务（Quests）、社区贡献、推荐等。

路线图（大致）：Q2 2026：AI Mining 上线、早期贡献者加入。
Q3 2026：OBN Tokenomics 发布、TGE（代币生成事件）、AI Mining 领取开启、初始上市。
Q4 2026 及以后：Staking 激活、移动端生态、AI 基础设施扩展。

空投 / 奖励信息项目明确有 早期参与奖励机制，通过积累 $OBN 来获得未来空投/分配：AI Mining：连接 EVM 钱包启动挖矿（持续积累）。
Quests/任务（部分奖励示例）：订阅 Newsletter：500 OBN
Follow X + 通知：500 OBN
加入 Telegram 群/频道：各 500 OBN
Like & Retweet 帖子：200-500 OBN 不等

其他：节点运行、社区贡献、推荐等可增加权重。
